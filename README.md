# Formación en Ciberseguridad – Fortinet

Este repositorio documenta las **certificaciones y el progreso completado** dentro del programa oficial de **Fortinet Network Security Expert (NSE)**.

Hace parte de mi ruta de formación en **Ciberseguridad**, como estudiante de **Ingeniería Informática**, con enfoque en fundamentos de seguridad, conceptos de red y concienciación en amenazas digitales.

---

## Cursos completados

###  Fortinet Certified Fundamentals in Cybersecurity
- **Estado:** Finalizado  
- **Certificación:** Obtenida  
- **Descripción:** Fundamentos de ciberseguridad, amenazas comunes, conceptos básicos de redes, protección de la información y concienciación en seguridad.

###  FCF – Getting Started in Cybersecurity 3.0
- **Estado:** Finalizado  
- **Certificación:** Obtenida  
- **Descripción:** Introducción estructurada a la ciberseguridad, roles, riesgos, ataques y principios defensivos.
